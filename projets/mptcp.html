Présentation du projet

Multipath TCP (MPTCP), développé à l'UCLouvain, est maintenant disponible sur Ubuntu 22.04. Cette extension à TCP permet aux utilisateurs d'utiliser simultanément les différentes interfaces réseau de leurs appareils (e.g. réseau câblé + Wi-Fi, ou encore Wi-Fi + 4/5G). Cela permet notamment d'augmenter le débit des connexions TCP ou de basculer d'un réseau à l'autre sans perdre la connexion. Le projet consiste à parcourir différentes applications utilisant le réseau, et de les mettre à jour pour les permettre d'utiliser le protocole MPTCP au lieu de TCP standard. Des candidats peuvent être le client SSR, OpenVPN, curl, les serveurs tels que Apache, Nginx, lighttpd, ou toute autre application réseau Open Source pouvant bénéficier de MPTCP.
